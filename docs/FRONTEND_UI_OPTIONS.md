# Варианты оформления фронтенда (WMS)

**Активная тема:** **D — тёмный «glass»** (полупрозрачные панели, `backdrop-filter`, светлые границы). На очень старых браузерах без размытия включается более плотный фон (см. `@supports` в `frontend/src/index.css`).

Если нужен другой стиль — напишите букву варианта или кратко опишите референс (сайт/скрин).

| Вариант | Название | Описание |
|--------|----------|----------|
| **A** | Улучшенный тёмный | Плотные поверхности без сильного blur (предыдущая версия до D). |
| **B** | Светлый корпоративный | Светлый фон (#f4f6f9), тёмный текст, синий акцент, чёткие границы. Похож на классические ERP/1С-веб. |
| **C** | Минимализм + моноширинный | Много воздуха, тонкие линии, акцент на данных; можно вернуть моноширинный шрифт для «техничного» вида. |
| **D** | Тёмный «glass» | Размытие (glassmorphism), полупрозрачные панели, «премиальный» SaaS-вид (чуть тяжелее для слабых ПК). |

**Рекомендация:** для склада при малых нагрузках на GPU — **D** выглядит современно; при тормозах — вернуть **A** или выбрать **B**.
